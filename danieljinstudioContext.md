## Design System

### Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Primary Background | White | `#FFFFFF` |
| Secondary Background | Off-white/Light gray | `#F5F5F5` |
| Text Primary | Near-black | `#1A1A1A` |
| Text Secondary | Gray | `#666666` |
| Accent Primary | Warm Beige/Sand | `#C4A77D` |
| Accent Alternative | Soft Coral/Terracotta | `#D4847C` |

### Typography

- **Headings:** Modern geometric sans-serif with personality (e.g., Inter, Satoshi, or General Sans)
- **Body:** Clean, highly readable sans-serif (e.g., Inter, DM Sans)
- **Hierarchy:** Strong contrast between heading weights (bold) and body (regular)

### Visual Style

- **Minimalism:** Generous whitespace, clean layouts
- **Glassmorphism:** Frosted glass effects on cards/modals
- **Neumorphism:** Subtle soft shadows for depth on interactive elements
- **Gradients:** Subtle, soft gradients (not harsh color transitions)
- **Animations:** Smooth, purposeful micro-interactions (Framer Motion)
- **3D Elements:** Subtle Three.js accents (floating shapes, particles, or abstract objects)

### Layout Principles

- Mobile-first responsive design
- Maximum content width: ~1200px
- Generous padding and margins
- Clear visual hierarchy
- Scroll-based reveal animations
